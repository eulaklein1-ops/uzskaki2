# IPOOLGO Uzbekistan — мобильный интернет-магазин

Готовый статический сайт для GitHub Pages.

## Что внутри
- mobile-first дизайн под трафик Instagram / TikTok;
- каталог с вкладками и отдельными карточками товаров;
- крупная галерея фото в карточке товара, свайп и полноэкранный просмотр;
- корзина и оформление заказа через WhatsApp;
- цены на круглые и прямоугольные бассейны под Узбекистан;
- новые разделы каталога: вкапываемые бассейны, купели и джакузи, SUP мебель, крышки и тенты, оборудование;
- локальные изображения в папках `assets/`, внешние CDN-ссылки не нужны;
- раздел FAQ / установка / технология / видео.

## Где менять товары и цены
Все товары находятся в файле:

```text
products.js
```

Для новых товаров, где нет твоей цены в сум, стоит `price: 0`, и на сайте показывается «Цена по запросу». Когда цена будет известна, просто замени `0` на нужную сумму.

## Как залить на GitHub Pages
1. Распакуй архив.
2. Загрузи все файлы в репозиторий GitHub.
3. Включи Pages: Settings → Pages → Deploy from branch → main / root.
4. Через несколько минут сайт откроется по ссылке GitHub Pages.
